# Frontend-Assignment

## Project Overview
Swift Shop Vendor Dashboard is a responsive E-commerce Vendor Portal built with **React 19** and **Tailwind CSS v4**. It allows merchants to:

- Manage product catalog
- Track inventory status
- Filter and search products dynamically

## How to Run
1. Clone the repository
2. change the directory and Install dependencies:
   ```bash
   npm install
3. Run the project
    ```bash
   npm run dev
